# <a name="tooltip"></a>ToolTip
Eine QuickInfo ist ein kleines Popup Fenster, das angezeigt wird, wenn ein Benutzer den Mauszeiger über ein Element, z. b. über einen, hält <xref:System.Windows.Controls.Button> .
