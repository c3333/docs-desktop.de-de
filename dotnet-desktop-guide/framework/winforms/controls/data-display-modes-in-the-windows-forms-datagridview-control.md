## <a name="virtual"></a><span data-ttu-id="411dd-129">Virtuell</span><span class="sxs-lookup"><span data-stu-id="411dd-129">Virtual</span></span>  
 <span data-ttu-id="411dd-130">Mit dem virtuellen Modus können Sie eigene Daten Verwaltungsvorgänge implementieren.</span><span class="sxs-lookup"><span data-stu-id="411dd-130">With virtual mode, you can implement your own data management operations.</span></span> <span data-ttu-id="411dd-131">Dies ist erforderlich, um die Werte ungebundener Spalten im gebundenen Modus beizubehalten, wenn das Steuerelement nach gebundenen Spalten sortiert wird.</span><span class="sxs-lookup"><span data-stu-id="411dd-131">This is necessary to maintain the values of unbound columns in bound mode when the control is sorted by bound columns.</span></span> <span data-ttu-id="411dd-132">Der primäre Einsatz des virtuellen Modus ist jedoch die Optimierung der Leistung bei der Interaktion mit großen Datenmengen.</span><span class="sxs-lookup"><span data-stu-id="411dd-132">The primary use of virtual mode, however, is to optimize performance when interacting with large amounts of data.</span></span>  
  
 <span data-ttu-id="411dd-133">Sie fügen das <xref:System.Windows.Forms.DataGridView> Steuerelement an einen von Ihnen verwalteten Cache an, und der Code steuert, wann Daten Zeilen per pushübertragung und-Pull abgerufen werden.</span><span class="sxs-lookup"><span data-stu-id="411dd-133">You attach the <xref:System.Windows.Forms.DataGridView> control to a cache that you manage, and your code controls when data rows are pushed and pulled.</span></span> <span data-ttu-id="411dd-134">Um den Speicherbedarf gering zu halten, sollte der Cache der Größe entsprechend der Anzahl der derzeit angezeigten Zeilen entsprechen.</span><span class="sxs-lookup"><span data-stu-id="411dd-134">To keep the memory footprint small, the cache should be similar in size to the number of rows currently displayed.</span></span> <span data-ttu-id="411dd-135">Wenn der Benutzer einen Bildlauf in neue Zeilen durchführt, fordert der Code neue Daten aus dem Cache an und leert optional alte Daten aus dem Arbeitsspeicher.</span><span class="sxs-lookup"><span data-stu-id="411dd-135">When the user scrolls new rows into view, your code requests new data from the cache and optionally flushes old data from memory.</span></span>
