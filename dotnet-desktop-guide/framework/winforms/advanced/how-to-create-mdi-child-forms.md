# <a name="how-to-create-mdi-child-forms"></a><span data-ttu-id="44f1c-103">Gewusst wie: Erstellen von untergeordneten MDI-Formularen</span><span class="sxs-lookup"><span data-stu-id="44f1c-103">How to: Create MDI child forms</span></span>

<span data-ttu-id="44f1c-104">Untergeordnete MDI-Formulare sind ein wesentliches Element von [MDI-Anwendungen (Multiple Document Interface)](multiple-document-interface-mdi-applications.md), da diese Formen den Mittelpunkt der Benutzerinteraktion bilden.</span><span class="sxs-lookup"><span data-stu-id="44f1c-104">MDI child forms are an essential element of [Multiple-Document Interface (MDI) applications](multiple-document-interface-mdi-applications.md), as these forms are the center of user interaction.</span></span>

<span data-ttu-id="44f1c-105">Im folgenden Verfahren verwenden Sie Visual Studio, um ein untergeordnetes MDI-Formular zu erstellen, in dem ein- <xref:System.Windows.Forms.RichTextBox> Steuerelement ähnlich wie bei den meisten Textverarbeitungsanwendungen angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="44f1c-105">In the following procedure, you'll use Visual Studio to create an MDI child form that displays a <xref:System.Windows.Forms.RichTextBox> control, similar to most word-processing applications.</span></span> <span data-ttu-id="44f1c-106">Indem Sie das Steuerelement durch <xref:System.Windows.Forms> andere Steuerelemente ersetzen, z. b. das-Steuerelement <xref:System.Windows.Forms.DataGridView> oder eine Kombination von Steuerelementen, können Sie untergeordnete MDI-Fenster (und durch Erweiterung MDI-Anwendungen) mit unterschiedlichen Möglichkeiten erstellen.</span><span class="sxs-lookup"><span data-stu-id="44f1c-106">By substituting the <xref:System.Windows.Forms> control with other controls, such as the <xref:System.Windows.Forms.DataGridView> control, or a mixture of controls, you can create MDI child windows (and, by extension, MDI applications) with diverse possibilities.</span></span>

## <a name="create-mdi-child-forms"></a><span data-ttu-id="44f1c-107">Erstellen von untergeordneten MDI-Formularen</span><span class="sxs-lookup"><span data-stu-id="44f1c-107">Create MDI child forms</span></span>

1. <span data-ttu-id="44f1c-108">Erstellen Sie ein neues Windows Forms-Anwendungsprojekt in Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="44f1c-108">Create a new Windows Forms application project in Visual Studio.</span></span> <span data-ttu-id="44f1c-109">Legen Sie im Fenster **Eigenschaften** für das Formular seine <xref:System.Windows.Forms.Form.IsMdiContainer%2A> -Eigenschaft auf `true` und die- `WindowsState` Eigenschaft auf fest `Maximized` .</span><span class="sxs-lookup"><span data-stu-id="44f1c-109">In the **Properties** window for the form, set its <xref:System.Windows.Forms.Form.IsMdiContainer%2A> property to `true` and its `WindowsState` property to `Maximized`.</span></span>

   <span data-ttu-id="44f1c-110">Dies kennzeichnet das Formular als MDI-Container für untergeordnete Fenster.</span><span class="sxs-lookup"><span data-stu-id="44f1c-110">This designates the form as an MDI container for child windows.</span></span>

2. <span data-ttu-id="44f1c-111">Ziehen Sie ein <xref:System.Windows.Forms.MenuStrip>-Steuerelement aus der `Toolbox` in das Formular.</span><span class="sxs-lookup"><span data-stu-id="44f1c-111">From the `Toolbox`, drag a <xref:System.Windows.Forms.MenuStrip> control to the form.</span></span> <span data-ttu-id="44f1c-112">Legen Sie die- `Text` Eigenschaft auf **File** fest.</span><span class="sxs-lookup"><span data-stu-id="44f1c-112">Set its `Text` property to **File**.</span></span>

3. <span data-ttu-id="44f1c-113">Klicken Sie auf die Schaltfläche mit den Auslassungs Zeichen (...) neben der Eigenschaft **Items** , und klicken Sie auf **Hinzufügen** , um zwei untergeordnete Menü Elemente für Tool</span><span class="sxs-lookup"><span data-stu-id="44f1c-113">Click the ellipsis (…) next to the **Items** property, and click **Add** to add two child tool strip menu items.</span></span> <span data-ttu-id="44f1c-114">Legen `Text` Sie die-Eigenschaft für diese Elemente auf **neu** und **Window** fest.</span><span class="sxs-lookup"><span data-stu-id="44f1c-114">Set the `Text` property for these items to **New** and **Window**.</span></span>

4. <span data-ttu-id="44f1c-115">Klicken Sie im **Projektmappen-Explorer** mit der rechten Maustaste auf das Projekt, und wählen Sie dann **Hinzufügen** > **Neues Element** aus.</span><span class="sxs-lookup"><span data-stu-id="44f1c-115">In **Solution Explorer**, right-click the project, and then select **Add** > **New Item**.</span></span>

5. <span data-ttu-id="44f1c-116">Wählen Sie im Dialogfeld **Neues Element hinzufügen** die Option **Windows Form** (in Visual Basic oder in Visual c#) oder **Windows Forms Anwendung (.net)** (in Visual C++) aus dem Bereich **Vorlagen** aus.</span><span class="sxs-lookup"><span data-stu-id="44f1c-116">In the **Add New Item** dialog box, select **Windows Form** (in Visual Basic or in Visual C#) or **Windows Forms Application (.NET)** (in Visual C++) from the **Templates** pane.</span></span> <span data-ttu-id="44f1c-117">Benennen Sie im Feld **Name** das Formular **Form2**.</span><span class="sxs-lookup"><span data-stu-id="44f1c-117">In the **Name** box, name the form **Form2**.</span></span> <span data-ttu-id="44f1c-118">Wählen Sie **Öffnen** aus, um das Formular dem Projekt hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="44f1c-118">Select **Open** to add the form to the project.</span></span>

    > [!NOTE]
    > <span data-ttu-id="44f1c-119">Das untergeordnete MDI-Formular, das Sie in diesem Schritt erstellten haben, ist ein Standard-Windows Form.</span><span class="sxs-lookup"><span data-stu-id="44f1c-119">The MDI child form you created in this step is a standard Windows Form.</span></span> <span data-ttu-id="44f1c-120">Als solches besitzt es eine Eigenschaft <xref:System.Windows.Forms.Form.Opacity%2A>, mit der Sie die Transparenz des Formulars steuern können.</span><span class="sxs-lookup"><span data-stu-id="44f1c-120">As such, it has an <xref:System.Windows.Forms.Form.Opacity%2A> property, which enables you to control the transparency of the form.</span></span> <span data-ttu-id="44f1c-121">Allerdings wurde die Eigenschaft <xref:System.Windows.Forms.Form.Opacity%2A> für Fenster der obersten Ebene entwickelt.</span><span class="sxs-lookup"><span data-stu-id="44f1c-121">However, the <xref:System.Windows.Forms.Form.Opacity%2A> property was designed for top-level windows.</span></span> <span data-ttu-id="44f1c-122">Verwenden Sie sie nicht mit untergeordneten MDI-Formularen, da Darstellungsprobleme auftreten können.</span><span class="sxs-lookup"><span data-stu-id="44f1c-122">Do not use it with MDI child forms, as painting problems can occur.</span></span>

     <span data-ttu-id="44f1c-123">Dieses Formular wird die Vorlage für Ihre untergeordneten MDI-Formulare.</span><span class="sxs-lookup"><span data-stu-id="44f1c-123">This form will be the template for your MDI child forms.</span></span>

     <span data-ttu-id="44f1c-124">Der **Windows Forms-Designer** wird geöffnet und zeigt **Form2** an.</span><span class="sxs-lookup"><span data-stu-id="44f1c-124">The **Windows Forms Designer** opens, displaying **Form2**.</span></span>

6. <span data-ttu-id="44f1c-125">Ziehen Sie ein **RichTextBox** -Steuerelement aus der **Toolbox** auf das Formular.</span><span class="sxs-lookup"><span data-stu-id="44f1c-125">From the **Toolbox**, drag a **RichTextBox** control to the form.</span></span>

7. <span data-ttu-id="44f1c-126">Legen Sie im Fenster **Eigenschaften** die `Anchor` -Eigenschaft auf **oben, Links** und die- `Dock` Eigenschaft auf **Fill** fest.</span><span class="sxs-lookup"><span data-stu-id="44f1c-126">In the **Properties** window, set the `Anchor` property to **Top, Left** and the `Dock` property to **Fill**.</span></span>

   <span data-ttu-id="44f1c-127">Dies bewirkt, dass das <xref:System.Windows.Forms.RichTextBox>-Steuerelement den Bereich des untergeordneten MDI-Formulars selbst dann vollständig ausgefüllt, wenn die Größe des Formulars geändert wird.</span><span class="sxs-lookup"><span data-stu-id="44f1c-127">This causes the <xref:System.Windows.Forms.RichTextBox> control to completely fill the area of the MDI child form, even when the form is resized.</span></span>

8. <span data-ttu-id="44f1c-128">Doppelklicken Sie auf das **neue** Menü Element, um einen- <xref:System.Windows.Forms.Control.Click> Ereignishandler dafür zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="44f1c-128">Double click the **New** menu item to create a <xref:System.Windows.Forms.Control.Click> event handler for it.</span></span>

9. <span data-ttu-id="44f1c-129">Fügen Sie Code ähnlich dem folgenden ein, um ein neues untergeordnetes MDI-Formular zu erstellen, wenn der Benutzer auf das **neue** Menü Element klickt.</span><span class="sxs-lookup"><span data-stu-id="44f1c-129">Insert code similar to the following to create a new MDI child form when the user clicks the **New** menu item.</span></span>

   > [!NOTE]
   > <span data-ttu-id="44f1c-130">Im folgenden Beispiel verarbeitet der Ereignishandler das <xref:System.Windows.Forms.Control.Click>-Ereignis für `MenuItem2`.</span><span class="sxs-lookup"><span data-stu-id="44f1c-130">In the following example, the event handler handles the <xref:System.Windows.Forms.Control.Click> event for `MenuItem2`.</span></span> <span data-ttu-id="44f1c-131">Beachten Sie, dass das **neue** Menü Element abhängig von den Besonderheiten ihrer Anwendungsarchitektur möglicherweise nicht ist `MenuItem2` .</span><span class="sxs-lookup"><span data-stu-id="44f1c-131">Be aware that, depending on the specifics of your application architecture, your **New** menu item may not be `MenuItem2`.</span></span>

    ```vb

       Dim NewMDIChild As New Form2()
       'Set the Parent Form of the Child window.
       NewMDIChild.MdiParent = Me
       'Display the new form.
       NewMDIChild.Show()
    End Sub
    ```

    ```csharp
    protected void MDIChildNew_Click(object sender, System.EventArgs e){
       Form2 newMDIChild = new Form2();
       // Set the Parent Form of the Child window.
       newMDIChild.MdiParent = this;
       // Display the new form.
       newMDIChild.Show();
    }
    ```

    ```cpp
    private:
       void menuItem2_Click(System::Object ^ sender,
          System::EventArgs ^ e)
       {
          Form2^ newMDIChild = gcnew Form2();
          // Set the Parent Form of the Child window.
          newMDIChild->MdiParent = this;
          // Display the new form.
          newMDIChild->Show();
       }
    ```

   <span data-ttu-id="44f1c-132">Fügen Sie in C++ `#include` am Anfang von Form1. h die folgende Direktive hinzu:</span><span class="sxs-lookup"><span data-stu-id="44f1c-132">In C++, add the following `#include` directive at the top of Form1.h:</span></span>


    <span data-ttu-id="44f1c-134">Dadurch kann das **Fenster** Menü eine Liste mit geöffneten untergeordneten MDI-Fenstern mit einem Häkchen neben dem aktiven untergeordneten Fenster verwalten.</span><span class="sxs-lookup"><span data-stu-id="44f1c-134">This enables the **Window** menu to maintain a list of open MDI child windows with a check mark next to the active child window.</span></span>
